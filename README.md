# Pacman


1. Erstellen eines neuen Objektes

1.1 Erstellen einer neuen Klasse für das Objekt.

1.2 Standardmethoden wie z.B. getX(), getY() oder setX(), setY(), aber auch evtl. collide().

1.3 Ein neues Objekt muss dann in der LevelDatei z.B. Level-1.txt hinzugefügt werden.

1.4 Danach muss dieses Object interpretiert werden von z.B. LevelDesign.java.

1.5 Anschließend muss in der Datei, die das Spielbrett darstellt, vereinbart werden wie das neue Objekt dargestellt werden soll.
